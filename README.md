# IDOR Research & Bug Bounty Workspace

This workspace is organized for tracking Insecure Direct Object Reference (IDOR) vulnerability research, proof-of-concept development, and bug bounty submissions.

## Folder Structure

```
IDOR/
├── reports/         # Bug bounty reports by status
│   ├── drafts/      # Work-in-progress reports
│   ├── submitted/   # Reports sent to programs
│   ├── accepted/    # Confirmed/rewarded reports
│   └── rejected/    # Declined or informative reports
├── poc/             # Proof of concept materials
│   ├── scripts/     # Automation and exploit scripts
│   ├── requests/    # Raw HTTP request samples
│   └── screenshots/ # Visual evidence
├── targets/         # Target application research
│   ├── recon/       # Reconnaissance notes
│   ├── scope/       # Program scope definitions
│   └── notes/       # General research notes
├── tools/           # Custom tools and utilities
├── templates/       # Reusable report templates
└── resources/
    └── references/  # External references, CVEs, writeups
```
