# Screenshots
Store visual PoC evidence here.

## Naming Convention
```
IDOR-[target]-[endpoint]-[step].png
```

## Examples
```
IDOR-example-billing-attacker-request.png
IDOR-example-billing-victim-response.png
IDOR-example-billing-enumeration.png
```

## Tips
- Use Burp Suite → right-click → "Save screenshot"
- Use browser DevTools → Network → right-click → "Copy as cURL" and screenshot
- Record video with OBS Studio for complex multi-step PoCs
- Annotate screenshots with arrows/boxes to highlight the IDOR evidence
